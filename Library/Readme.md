# Web-Приложение Library

Простое приложение для хранения информации о книгах. 

* ## stack:
  + Python3;
  + Django 4;
  + Вместо стандартной БД SQlite3 - XLSX файл;
  + Jinja;
  + Bootstrap 5;
  + Excel (openpyxl). Разбивка (пагинация) по 5 постов;
